# java-codes Java Core Programs

This repository contains Java Core programs that demonstrate the fundamental concepts of Java programming. It is designed for students, beginners, and anyone preparing for coding interviews or university practicals.

Topics Covered

- Variables and Data Types
- Operators
- Conditional Statements ("if", "if-else", "switch")
- Loops ("for", "while", "do-while")
- Arrays
- Strings
- Methods
- Classes and Objects
- Constructors
- Inheritance
- Polymorphism
- Abstraction
- Encapsulation
- Exception Handling
- File Handling
- Collections Framework
- Multithreading
- Basic Searching and Sorting Programs

Requirements

- Java JDK 8 or later
- VS Code, Eclipse, or IntelliJ IDEA

How to Run

1. Clone the repository:

git clone https://github.com/your-username/your-repository.git

2. Open the project in your Java IDE.

3. Compile and run any Java file:

javac FileName.java
java FileName
